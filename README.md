# Shambala NYC static website

Bilingual static website template for shambala.nyc.

## Included
- Home
- About us
- Programs
- Events
- Contacts
- RU/EN language switcher
- Responsive menu
- Donate button
- Floating chat contact widget
- Google Maps block

## Replace before publishing
- PayPal donate link
- Instagram link
- Telegram link
- Opening hours
- Event texts and photos
- Any real organization text
